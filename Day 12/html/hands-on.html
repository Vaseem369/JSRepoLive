safe JSON, guarded setAge, ValidationError, and a tiny ES-module pair ( mathUtils.js + app.js).

Task 1 Safe JSON parse

Implement safeParse(str) with try/catch around JSON.parse(str): success → return the value; failure → log "Invalid JSON:" plus the error message and return null. Test with valid and broken JSON strings.
