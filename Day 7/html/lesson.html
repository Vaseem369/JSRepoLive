Lesson

Day 7 — Arrays

Indexing, mutating vs non-mutating methods, forEach / map, predicates, reduce, and the spread operator — with reference output for every topic.

Topic 1 Creating & Accessing — index 0, length, .at(-1)

Example code 

const fruits = ["apple", "banana", "mango"];

console.log(fruits[0]);                   // apple
console.log(fruits[2]);                   // mango
console.log(fruits.length);               // 3
console.log(fruits[fruits.length - 1]);   // mango (last)

// Modify an element
fruits[1] = "cherry";
console.log(fruits);                      // changed array

// Negative index? No — undefined
console.log(fruits[-1]);                  // undefined  (use .at(-1) instead)
console.log(fruits.at(-1));               // mango
