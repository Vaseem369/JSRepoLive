Hands-on

Day 7 — Arrays

Cart mutations, filter/find/every/some on scores, map with GST (and rounding), then reduce for totals and max.

Task 1 Cart Manipulation

Start with const cart = ["bread", "milk", "eggs"];. Use push to add "butter", then unshift to add "rice" at the start. Log the cart.

Use pop to remove the last item. Log what was removed and the new cart.

Use splice(1, 1) to remove one item starting at index 1. Log the result.
