# JavaScript Basics Learning Repository

A beginner-friendly JavaScript learning repository covering core JavaScript concepts with structured lessons, hands-on exercises, and homework tasks.

---

# 📚 Course Structure

This repository is organized day-wise to help learners gradually build JavaScript knowledge from fundamentals to advanced basics.

Each day contains:

- `lesson.html` → Concept explanations
- `hands-on.html` → Practice examples
- `homework.html` → Exercises and assignments
- `js/` → JavaScript source files
