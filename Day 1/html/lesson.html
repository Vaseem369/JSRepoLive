Topic 1 What is JavaScript? Browsers & beyond

JavaScript was created in 1995 by Brendan Eich at Netscape — legend says the first version shipped in about 10 days. Despite the name, it is not related to Java.

The language is standardised as ECMAScript; engines (Chrome’s V8, Firefox’s SpiderMonkey, Safari’s JavaScriptCore) implement that spec. Today JavaScript runs in three big places:

  • Browsers — almost every interactive webpage.
  • Servers — e.g. Node.js.
  • Apps & desktops — e.g. React Native, Electron; VS Code itself is built with web tech + JS.
