Day 5 — Loops & Iteration

for, while / do...while, break / continue, for...of and for...in — with reference code and output for each task.

Task 1 Multiplication Table

Use a for loop to print the multiplication table of 7 from 1 to 10. Format: 7 x 1 = 7, …, 7 x 10 = 70. Use template literals for clean strings.

Bonus: print only the even multiples of 7 (use continue or change the step).
