Hands-on

Day 2 — Primitive Types

In-class tasks for practicing primitive types, template literals, null vs undefined, and number quirks in JavaScript.

Task 1 typeof Exploration

In the console, log typeof for: 42, "hello", true, undefined, null, 100n, Symbol("x").

Note which one returns a SURPRISING type.

In a comment, write down which one and why.

                        Task 2
                    

                    
                        Template Literals
                    

                

                


                    

                        Create variables:
                        const name = "Aarav",
                        const age = 22,
                        const city = "Jaipur".
                    


                    

                        Build the string
                        "Aarav, 22, from Jaipur"
                        using OLD-style concatenation with +.
                    


                    

                        Build the same string using
                        a TEMPLATE LITERAL.
                    


                    

                        Build a 3-line "card"
                        using a multi-line template literal:
                    


                    

                        Line 1: Name: <name>
                    


                    

                        Line 2: Age: <age>
                    


                    

                        Line 3: City: <city>
                    


                    

                        Reference solution output
                    


                    


                


            


            

            


                

                    
                        Task 3
                    

                    
                        null vs undefined
                    

                

                


                    

                        Declare let user; (no value).
                        Log it and log its typeof.
                    


                    

                        Set user = null.
                        Log it and log its typeof.
                    


                    

                        Compare:
                        null == undefined
                        and
                        null === undefined.
                    


                    

                        In a comment, explain when you should use
                        null vs leaving it undefined.
                    


                    

                        Reference solution output
                    


                    


                


            


            

            


                

                    
                        Bonus
                    

                    
                        Number Quirks
                    

                

                


                    

                        Compute 0.1 + 0.2
                        and log the result.
                    


                    

                        Compare it to 0.3 using ===.
                    


                    

                        Try:
                        Number.isInteger(5),
                        Number.isInteger(5.0),
                        Number.isInteger(5.5).
                    


                    

                        Try:
                        1 / 0
                        and
                        "abc" * 2.
                    


                    

                        Note the special values shown.
