Day 6 — Functions

Declarations, return values, expressions, arrows, defaults, scope — with reference code and output for every task.

Task 1 Rectangle Area

Write function area(length, width) that returns the area. Call it three times with different values and log each result.

Bonus: rewrite the same logic as an arrow function with implicit return.
