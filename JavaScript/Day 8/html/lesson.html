Lesson

Day 8 — Objects

Literals, dot vs bracket, mutating properties on const, methods and this, destructuring, Object.keys / values / entries, spread and Object.assign — with reference output for each topic.

Topic 1 Object literals — { key: value, ... }

Syntax: const obj = { key1: value1, key2: value2 }; Values can be strings, numbers, booleans, arrays, nested objects, or functions (methods when you call them on the object).

Example 

const profile = {
    username: "riya_91",
    score: 1200,
    active: true,
    tags: ["js", "web"],
    prefs: { theme: "dark" },
    ping() {
        return "ok";
    }
};

                        Topic 2
                    

                    
                        Dot vs bracket notation
                    

                

                


                    

                        Dot
                        —
                        user.name
                        — readable default.
                        Bracket
                        —
                        user["name"]
                        — use for
                        dynamic
                        keys (variables) or keys that are not valid identifiers
                        (spaces, hyphens, numbers at start, etc.).
                    


                    

                        Example
                    


                    
const user = { name: "Priya", "full-name": "Priya S." };
const key = "name";

user.name;
user["name"];
user[key];
user["full-name"];

                        Topic 3
                    

                    
                        Add · Update · Delete (even with const)
                    

                

                


                    

                        const
                        fixes the
                        binding
                        to the object, not the object's contents — you can add,
                        change, and delete properties freely.
                    


                    

                        Example
                    


                    
const user = { name: "Priya", age: 24 };

user.email = "priya@example.com";
user.city = "Jaipur";
user.age = 25;
delete user.city;

console.log(user);


                    

                        Reference output
                    


                    


                


            


            

            


                

                    
                        Topic 4
                    

                    
                        Methods & this
                    

                

                


                    

                        A property whose value is a function behaves like a
                        method
                        when you call it as
                        obj.method().
                        Inside that function,
                        this
                        is the object the method was invoked on (with caveats
                        later for loose functions and arrows).
                    


                    

                        Example
                    


                    
const counter = {
    n: 0,
    bump() {
        this.n += 1;
        return this.n;
    }
};

counter.bump();
counter.bump();


                    

                        Reference output
                    


                    


                


            


            

            


                

                    
                        Topic 5
                    

                    
                        Destructuring
                    

                

                


                    

                        Pull properties into variables:
                        const { name, age } = user;
                        Rename with
                        name: userName,
                        defaults with
                        phone = "N/A",
                        and use the same pattern in function parameters.
                    


                    

                        Example
                    


                    
const user = {
    name: "Riya",
    age: 23,
    city: "Jaipur",
    email: "r@x.in"
};

const { name, age } = user;
const { name: userName, city } = user;
const { phone = "N/A" } = user;

const greet = ({ name, age }) => `Hi ${name}, age ${age}`;
greet(user);


                    

                        Reference output
                    


                    


                


            


            

            


                

                    
                        Topic 6
                    

                    
                        Object.keys · values · entries
                    

                

                


                    

                        Quick reference
                    


                    
Object.keys(obj)     → array of keys
Object.values(obj)   → array of values
Object.entries(obj)  → array of [key, value] pairs


                    

                        Reference output
                    


                    


                


            


            

            


                

                    
                        Topic 7
                    

                    
                        Object.assign & spread
                    

                

                


                    

                        Object.assign(target, ...sources)
                        copies enumerable own properties into
                        target.
                        Object spread
                        { ...a, ...b }
                        is the usual way to merge or shallow-copy objects today.
                    


                    

                        Example
                    


                    
const a = { name: "Priya" };
const b = { age: 24 };

const merged = Object.assign({}, a, b);
const merged2 = { ...a, ...b };
const copy = { ...a };
