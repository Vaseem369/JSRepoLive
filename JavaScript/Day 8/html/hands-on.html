Hands-on

Day 8 — Objects

Build and mutate a student object, a bankAccount with this, destructuring on a product — plus iterable object bonuses.

Task 1 Build a student object

Create student with name = "Anaya", age 21, city "Jaipur", course "B.Tech", and marks (an array of three numbers). Log the object, then log name, age, and the first mark using dot notation.

Add email = "anaya@example.com", set age to 22, and delete city. Log the student again.
