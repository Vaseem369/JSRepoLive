Hands-on

Day 3 — Operators & Core JS Concepts

Practice GST math with template literals, compare == vs ===, use logical defaults, and chain ternaries with care.

Task 1 GST Calculator

Take const price = 850. Compute the GST at 18% and the total (price + GST).

Use a template literal to log: "Price: ₹850, GST: ₹153, Total: ₹1003" (match your computed numbers).

Try price = 1234.50. Note any floating-point oddness in the logged values.

                        Logical Operators
                    

                

                


                    

                        Take
                        const age = 22,
                        const hasEmail = true.
                        Build
                        canRegister
                        using
                        &&

                        hasEmail.
                    


                    

                        Take
                        const userName = "".
                        Use
                        ||
                        to default it to
                        "Guest".

                        !
                        to flip a boolean:
                        const isLoggedIn = false;

                        const isLoggedOut = !isLoggedIn.
